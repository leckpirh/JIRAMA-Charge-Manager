Guide d'utilisation

×

📘 Bienvenue dans JIRAMA Charge Manager

Ce guide vous aidera à utiliser toutes les fonctionnalités de l'application.

1. Ajouter des colocataires

Allez dans l'onglet "Colocataires" → "Ajouter un colocataire". Renseignez ses informations et son coefficient de présence.

2. Ajouter des appareils électriques

Allez dans l'onglet "Appareils" → "Ajouter un appareil". Indiquez sa puissance, ses heures d'utilisation et son type (individuel/partagé).

3. Saisir les factures JIRAMA

Allez dans "Paramètres" → renseignez les montants de vos factures d'électricité et d'eau.

4. Visualiser la facturation

L'onglet "Facturation" calcule automatiquement la répartition des charges entre colocataires.

5. Générer une facture PDF

Cliquez sur "Générer PDF" pour obtenir une facture officielle au format PDF.

6. Suivre l'historique

L'onglet "Historique" conserve toutes vos factures et paiements.

Astuce : Utilisez le coefficient de présence pour ajuster la part des colocataires absents !

Foire aux questions

×
Comment fonctionne le coefficient de présence ?

Le coefficient permet d'ajuster la part des charges selon la présence réelle. Exemple : un colocataire présent 50% du mois aura un coefficient de 0.5. Cela permet de ne pas payer pour les périodes d'absence.

Différence entre appareil individuel et partagé ?

Appareil individuel : Facturé uniquement à la personne qui l'utilise (ex: ordinateur personnel).
Appareil partagé : Réparti équitablement entre tous les colocataires selon leur coefficient de présence (ex: réfrigérateur, télévision).

Comment importer mes factures JIRAMA ?

Rendez-vous dans l'onglet "Paramètres" et saisissez manuellement le montant de vos factures d'électricité et d'eau. L'application calculera automatiquement la répartition entre les colocataires.

Puis-je exporter mes données ?

Oui ! Utilisez le bouton "Exporter" dans le Dashboard pour sauvegarder toutes vos données au format JSON. Vous pourrez les réimporter plus tard avec le bouton "Importer".

Comment générer une facture PDF ?

Dans l'onglet "Facturation", cliquez sur "Générer PDF". Une facture professionnelle sera automatiquement téléchargée avec le détail des charges par personne.

Les données sont-elles sauvegardées ?

Oui ! Toutes vos données sont automatiquement sauvegardées dans votre navigateur (localStorage). Vous pouvez aussi les exporter pour les conserver en sécurité.
